## Description

The `faucet_testnet_plugin` provides an interface that assists in the automation of distributing tokens on an INERY testnet.

## Usage

```console
# config.ini
plugin = inery::faucet_testnet_plugin
```
```sh
# command-line
nodine ... --plugin inery::faucet_testnet_plugin
```

## Options

None
